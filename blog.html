1. The difference between map(),filter(),find() and forEach() ?

→ map() is used for transforming each element in an array and returning a new array with the same length.
→ filter() is used for creating a new array with only the elements that meet a certain criteria.
→ find() returns the first element in an array that satisfies a given condition. It returns undefined if no element satisfies the condition. find() stops searching the array as soon as it finds an element that satisfies the condition. Once the condition is met, find() does not continue searching for other elements that may satisfy the condition. find() returns only one element that meets the condition.
→ forEach() is used for executing a function on each element in an array without returning a new array

2. The difference Local Storage, Session Storage, Cookie ?

→ Local storage provides a larger storage space compared to cookies, and data stored in local storage remains available even after the user closes the browser or turns off their device. Session storage is similar to local storage,but automatically cleared when the browser is closed. Cookies are used for storing small amounts of data and they can be accessed by any page from the same domain.
